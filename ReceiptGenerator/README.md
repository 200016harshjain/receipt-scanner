# ReceiptGenerator

> ### Install:
> Download or Clone repo;  
> In terminal, in ReceiptGenerator directory:  
> `pip install -r requirements.txt`
---
> ### Usage:
> In terminal, in ReceiptGenerator directory:


> `python run.py {N}`  
> Creates N sample receipt sets.  
> Example:  
> `python run.py 3`


> `python run.py clear`  
> Clears the whole result folder.  
> Example:  
> `python run.py clear`
